# Express Coders X

All the demo code and exercises of Express Node JS

## Installation

Use the package manager [pub](https://pugjs.org/api/getting-started.html) to install for lessons 1 and 2.

```bash
$ npm install pug

```

## Usage

```
var express = require('express');
var app = express();
var port = 3000;
app.set('view engine', 'pug');
app.set('views', './views');

```
